# 🎭 Kalavidara-Balaga
### Folk Artist Talent Hub

Kalavidara-Balaga is an Android application developed to support and promote Karnataka’s traditional folk artists such as Dollu Kunitha and Pooja Kunitha performers.

The application acts as a digital talent marketplace connecting rural folk artists with urban event planners, wedding organizers, and cultural event managers.

---

# 📱 Project Overview

Traditional folk artists often depend only on seasonal events for income. Many talented performers lack digital exposure and direct market access.

Kalavidara-Balaga solves this problem by providing:

- Artist digital profiles
- Talent discovery platform
- Performance galleries
- Direct booking support
- GenAI-generated descriptions
- Cultural preservation through technology
